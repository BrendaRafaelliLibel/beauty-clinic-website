# Beauty Clinic Website

This project is a simple beauty clinic website developed with HTML and CSS.  
It was created to practice front-end development, page structure, navigation, and styling.

## Features

- Home page
- Services page
- Contact page
- Navigation menu
- Service cards
- Appointment form
- Responsive layout structure

## Services

- Manicure
- Pedicure
- Hair Wash
- Blowout
- Hair Hydration
- Eyebrow Design

## Technologies

- HTML
- CSS
